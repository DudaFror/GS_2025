Público-alvo

O projeto utiliza sensores de baixo custo acoplados a microcontroladores Arduino para monitorar o nível da água em tempo real. Esses dispositivos capturam os dados e enviam para a plataforma web por meio de comunicação via internet. A plataforma é desenvolvida com HTML, CSS e JavaScript, oferecendo uma interface simples e responsiva para exibir alertas e informações atualizadas.

Os dados são armazenados na nuvem, garantindo segurança e disponibilidade constante. Essa combinação de hardware acessível e software eficiente torna o sistema escalável e fácil de implementar em diferentes regiões.
